# Endpoints

## Current user

- `GET /user` - Returns the currently authenticated user
- `GET /user/settings` - Returns all settings of the currently logged in user
- `GET /user/settings/{settingKey}` - Returns the given setting of the currently logged in user
- `PUT /user/settings/{settingKey}` - Sets the given setting of the currently logged in user
- `DELETE /user/settings/{settingKey}` - Deletes the given setting of the currently logged in user
